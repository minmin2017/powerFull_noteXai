# Powerfull Note 🧠✏️

แอปจดโน้ตแบบ **Mind Map** ที่:

- เห็นภาพรวมหัวข้อทั้งหมด แล้วซูม/แพนลงไปดูหัวข้อย่อยได้
- เพิ่ม/แก้/ย้ายโหนดด้วยเมาส์
- มี **โหมดวาดด้วยปากกา/เมาส์** (freehand รองรับแรงกดปากกา)
- มี **พาเนลแชท Claude ฝั่งซ้าย** ไว้อ่านสิ่งที่ Claude บอก
- มี **ปุ่มพูดภาษาไทย** (Web Speech API `th-TH`) — แก้ปัญหาที่ `/voice` ของ Claude Code ไม่รองรับไทย
- เชื่อมกับ **Claude Code ผ่าน MCP** — สั่งด้วยเสียง/ข้อความแล้วให้ Claude สร้างมายด์แมปให้

---

## วิธีเริ่มใช้

### 1) เปิดแอป

ดับเบิลคลิก **`start.cmd`** หรือรันในเทอร์มินัล:

```powershell
cd C:\Users\wicha\Desktop\powerfull_note
npm start
```

แล้วเปิด **http://localhost:4321** ใน **Chrome หรือ Edge** (จำเป็นสำหรับปุ่มเสียงไทย)

> ครั้งแรกเบราว์เซอร์จะขอสิทธิ์ไมโครโฟน — กดอนุญาต

### 2) ต่อ Claude Code (MCP)

เปิดเทอร์มินัลอีกอันแล้วรัน `claude` ใน **โฟลเดอร์เดียวกันนี้**:

```powershell
cd C:\Users\wicha\Desktop\powerfull_note
claude
```

ไฟล์ `.mcp.json` ในโฟลเดอร์จะถูกโหลดอัตโนมัติ — Claude Code จะถามให้อนุญาต MCP server ชื่อ
`powerfull-note` ให้กดอนุญาต (ตรวจได้ด้วยคำสั่ง `/mcp`)

> ⚠️ ต้องเปิดแอป (ข้อ 1) ค้างไว้ก่อน MCP ถึงจะทำงาน เพราะ MCP คุยกับเซิร์ฟเวอร์ของแอป

---

## ไหลการใช้งานที่ตั้งใจไว้

1. กดปุ่มไมค์ในแอป 🎤 → พูดภาษาไทย → ข้อความถูก **คัดลอกอัตโนมัติ**
2. เอาไปวางใน Claude Code **หรือ** บอก Claude ว่า "ดูเสียงล่าสุด" แล้ว Claude เรียก `get_voice_input` เอง
3. Claude สร้าง/จัดหัวข้อในมายด์แมปให้ (เห็นอัปเดตสดในแอป)
4. Claude พิมพ์สรุป/คำถามกลับมาที่ **พาเนลซ้าย** ผ่าน `say_to_user` ให้คุณอ่าน

---

## เครื่องมือ MCP ที่ Claude ใช้ได้

| เครื่องมือ | หน้าที่ |
|---|---|
| `get_mindmap` | อ่านโครงมายด์แมปปัจจุบัน (พร้อม id ของแต่ละโหนด) |
| `add_topic` | เพิ่มหัวข้อ/หัวข้อย่อย (ระบุ `parentId` เพื่อทำเป็นลูก) |
| `add_topics_bulk` | เพิ่มหลายหัวข้อพร้อมกัน + ผูกลูก-แม่ในคำสั่งเดียว |
| `update_node` | แก้ข้อความ/สีของโหนด |
| `move_node` | ย้ายตำแหน่งโหนด |
| `delete_node` | ลบโหนด (และลูกทั้งหมด) |
| `say_to_user` | ส่งข้อความเข้าพาเนลแชทซ้าย |
| `get_voice_input` | ดึงข้อความเสียง/ข้อความล่าสุดที่ผู้ใช้พูด/พิมพ์ |

ตัวอย่างคำสั่งกับ Claude Code:
> "ดูเสียงล่าสุดที่ฉันพูด แล้วแตกเป็นมายด์แมปหัวข้อย่อยให้หน่อย"

## การควบคุมในแอป

- **เพิ่มหัวข้อ**: ปุ่ม `+ หัวข้อ` บนแถบเครื่องมือ หรือ **ดับเบิลคลิก** ที่ว่าง ๆ
- **เพิ่มหัวข้อย่อย**: ชี้ที่โหนด แล้วกดปุ่ม `+` ทางขวาของโหนด
- **แก้ข้อความ**: ดับเบิลคลิกที่โหนด
- **ย้าย**: ลากโหนด • **เลือก/ลบ**: คลิกโหนดแล้วกด `Delete`
- **ยุบ/ขยายหัวข้อย่อย**: ปุ่มวงกลม `−/+` ทางซ้ายของโหนด
- **เลื่อนกระดาน**: ลากที่ว่าง (หรือกด Space ค้างแล้วลาก / คลิกขวาลาก)
- **ซูม**: สกอลล์เมาส์ • **ภาพรวม**: ปุ่ม `🔭 ภาพรวม`
- **วาดด้วยปากกา**: ปุ่ม `✏️ วาด` → เลือกสี/ขนาด • `🧽 ลบเส้น` คลิกลายเพื่อลบ • `↶ ย้อน`

## หมายเหตุ

- ข้อมูลถูกเก็บไว้ที่ `data/mindmap.json`
- พอร์ตเริ่มต้น `4321` — เปลี่ยนได้ด้วย env `PORT` (ต้องแก้ใน `.mcp.json` ด้วย)
- เรื่อง `/voice` ของ Claude Code: ตัวมันเป็นโปรแกรมปิด แก้ที่ตัวมันไม่ได้
  จึงทำปุ่มเสียงไทยไว้ในแอปนี้แทน ซึ่งใช้เครื่องมือถอดเสียงของ Chrome/Edge ที่รองรับไทยเต็มที่
